A Reference Implementation & Pattern Library

Paper captures the moment.
The terminal organizes the life.

A hybrid GTD system you can study, fork, and adapt. Built on two engines: a waterproof weekly sheet for daily capture, and a plain-markdown CLI for structured review across five horizons of commitment.

Not a turnkey app — a pattern library. Clone it, learn from it, make it your own.

Two Engines, One System

Paper is fast, tactile, and creative. Digital is searchable, reviewable, and hierarchical. Together they cover every altitude of your life.

The Weekly Sheet

  • ✓ Front: 7-day grid with daily tasks, weather, and temperature highs
  • ✓ Back: Inbox list, sketches, calligraphy practice, mind maps
  • ✓ Bottom row: Daily habit scores (SC, GE, RN, IP, DI, DC) rated 1–5
  • ✓ Multi-color ink: Red for completed items, blue for notes, black for tasks
  • ✓ Paper: iGage Weatherproof 10-mil, cut in half — waterproof, durable, Copic-friendly
  • ✓ Capture: Write anywhere, anytime — no battery required

The Terminal Tool

  • ✓ 5 horizon files: purpose.md, vision.md, goals.md, areas.md, projects.md
  • ✓ Rust CLI: Single binary, zero dependencies, works fully offline
  • ✓ Weekly review: Guided 4-step walkthrough with vim keybindings
  • ✓ AI integration: Accountability partner and brainstorm skills via pi coding agent
  • ✓ Plain markdown: Every file is human-readable, scriptable, versionable

The Weekly Review

Every week, a guided 4-step walkthrough keeps the system trusted. Run gtd review and follow the prompts.

Steps 0 and 3 integrate with the pi coding agent — spawning the accountability-partner and gtd-brainstorm skills for AI-powered review and ideation.

0

Assessment

Read last week's board. See your accountability partner score. Press p to spawn the pi accountability-partner skill for AI-powered performance feedback.

1

Get Clear

Process the calendar. Check waiting-for items. Clear the inbox to zero.

2

Get Current

Review projects, next actions, someday/maybe. Walk the trigger list. Check all five horizons.

3

Get Creative

Brainstorm new ideas. Press g to spawn the pi gtd-brainstorm skill and generate 30 fresh ideas tailored to your system.

Beyond the Horizons

Below the five horizons sit the ground-level lists that keep daily life running.

Calendar

Family birthdays, holy days, financial due dates, full moon names, seasonal events. Extends 18+ months out.

Waiting For

Items delegated or dependent on others: new dentist, IRA investment timing, completed obituary notification.

Trigger List

Categorized prompts for when you're stuck: home repairs, finances, faith, health, transportation, relationships.

Someday / Maybe

Parking lot for ideas that aren't active yet: Energization Exercises mastery, fitness cart for power blocks.

Checklists

Reference documents: diet plan with protein targets, 11 supplements, room-by-room maintenance, monthly budget, neighbor names.

Seed List

15 go-to next actions for low-energy moments: play guitar, go fishing, cut hair, attend Mass, study copperplate.

How Paper Feeds Digital

The paper sheet captures raw life. The digital system organizes it into horizons. The weekly review bridges them.

Why This Works

Paper removes friction. No login, no loading screen, no battery. A folded sheet fits in a pocket. The act of handwriting engages memory differently than typing.

Markdown removes lock-in. Every file is plain text. Version it with git. Edit it with vim. Feed it to an AI. Print it. The data is yours forever.

The terminal removes distraction. No notifications, no social feeds, no ads. Just your data and your keyboard, exactly how you configured it.

The weekly review removes drift. Every Sunday, the system asks you: are your daily actions aligned with your five horizons of commitment?

One binary. Markdown files. A folded sheet of paper.
